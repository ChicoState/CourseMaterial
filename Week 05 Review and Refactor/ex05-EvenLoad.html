Your Task

You are working on an application that manages shipping emergency aid kits. There are two trucks for delivering the kits and we want to distribute their weight evenly and also have both trucks travel to the same number of cities. The kits for the first half of the list will go on Truck A and the second half will go on Truck B. However, we also want Truck B to move some kits to a warehouse (in a different city), but without carrying more kits than Truck A and without delivering to more cities.

Write a function that receives an array of integers (of size n), where each element in the array represents the number of kits requested by a city. Return the number of kits that can be loaded on Truck B to ship to the warehouse, 0 if it should not deliver to the warehouse at all, or throw an exception if it is impossible to balance Truck A and Truck B with the same number of locations and same number of kits (without changing the order of the list). Keep in mind: Test-Driven Development and Pair Programming rules.
